# CalMse

## Installation
Please ensure that the project has the following structure.
```
CalMse/
├── build/
├── include/
├── res/
│   ├── CompMatchRes/
│   │   ├── tra
│   │   ├── rot
│   │   ├── lig_rect
│   │   └── lig_qbc
│   ...
├── .gitignore
├── CMakeLists.txt
├── main.cpp
└── README.md
```

Add the matching results of each algorithm to the "CalMse/res" path.

## Usage
After generating the project with CMake, it can be run directly.
